# app

Balance Life

## Project folder structure

- ```assets``` folder for image, icon
- ```lib``` folder for handling app logic
- ```tests``` folder for testing

### Folder tree

- lib/
    - main.dart
    - models/
        - food_model.dart
        - ingredient_model.dart
        - nutrient_model.dart
    - screens/
        - home_screen.dart
        - food_detail_screen.dart
        - nutrient_detail_screen.dart
    - widgets/
        - custom_button.dart
        - nutrient_card.dart
    - services/
        - api_service.dart
        - database_service.dart
    - utils/
        - constants.dart
    - theme/
        - app_theme.dart
    - routes.dart
    - app.dart
